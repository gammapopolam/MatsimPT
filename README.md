# MatsimPT

Matsim Preprocessing Tools - простые инструменты для анализа данных из MATSim и конкретно пакета Minibus.

## transitSchedule.py

Переводит сеть, линии ОТ в перевариваемый ГИС-пакетами вид, а также анализирует дневное наполнение каждой единицы техники.

## events.py

Переводит все события за день каждой единицей техники в перевариваемый ГИС-пакетами вид.

## plans.py 

Переводит все перемещения агентов-пассажиров за день в перевариваемый ГИС-пакетами вид.

## PTM_config_mod.xml

Образец конфига для маппинга GTFS в MATSim transitSchedule с помощью модуля pt2matsim

## default_minibus_config.xml

Образец конфига для запуска модели паратранзита MATSim.
